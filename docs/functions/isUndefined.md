# Function: isUndefined()

> **isUndefined**(`val`): `val is undefined`

## Parameters

• **val**: `unknown`

unknown - The value to check if it's undefined.

## Returns

`val is undefined`

## Description

If the value is undefined, return true, otherwise return false.

## Examples

```ts
isUndefined("string") // false
```

```ts
isUndefined() // true
```
