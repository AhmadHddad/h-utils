    Description

    Transform a value by piping it through the listed functions. Sugar syntax for pipe(f, g)(value).

    Example

    const add1 = (n: number) => n + 1;
    const times2 = (n: number) => n * 2;

    const result: number = pipeWith(1, add1, times2) // 4;
